4. pengertian traffic monitoring beserta 5 tools masing-masing tools mempunyai fungsi dan cara kerja. Mohon di jawab.Terima kasih


#Pengertian Traffic Monitoring:

Traffic Monitoring merupakan teknik monitoring jaringan dengan melihat paket aktual dari traffic pada jaringan dan menghasilkan laporan berdasarkan traffic jaringan.

#Tool-tool tersebut diantaranya sebagai berikut:

→Tools pendeteksi jaringan

Tool ini Memperhatikan beacon yang dikirim oleh akses point nirkabel, dan menampilkan informasi seperti nama jaringan, kekuatan sinyal yang didapat, dan channel.

→Tool spot check

Tool ini di desain untuk troubleshooting dan biasanya dikelola secara interaktif selama periode waktu yang singkat. Program seperti Ping mungkin dianggap sebagai tool spot check aktif, karena dia mengeluarkan trafik dan melakukan polling ke mesin tertentu. Tool spot check pasif termasuk protokol analyIer , yang memeriksa setiap paket di jaringan dan menyediakan perincian secara detail mengenai percakapan jaringan (termasuk alamat sumber dan tujuan, informasi protokol, dan bahan data aplikasi).

→ Tool trending

Tool ini Menjalankan monitor tanpa operator dalam periode lama, dan biasanya menyiapkan hasil menjadi grafik.

→Tool monitor realtime

Tool ini Menjalankan monitor yang sama, tetapi segera memberitahu administrator jika merekamengetahui masalah.

→Tool penguji throughput

Tool ini Memberitahu anda bandwidth sebenarnya yangada di antara dua ujung di jaringan.

→Tool Intrusion detection

Tool ini Mengamati trafik jaringan yangtidak diinginkan, dan mengambil keputusan yang tepat (biasanya menolak akses dan/atau memberitahu seorang network administrator).

→Tool benchmarking

Tool ini Memperkirakan kinerja maksimum dari sebuah layanan atau sambungan jaringan.

7. 1. apa yang disebut tools TIK, apa saja yang meliputi tools TIK dan sebutkan contoh dari masing masing tools !​


Jawaban:

Tools TIK adalah semua perangkat lunak, perangkat keras, dan aplikasi yang digunakan untuk memproses, mengelola, dan menyimpan informasi secara digital. Tools TIK dapat membantu pengguna dalam menyelesaikan tugas-tugas pekerjaan, meningkatkan produktivitas dan efisiensi, serta mempermudah dalam mengakses informasi. Beberapa contoh tools TIK diantaranya adalah:

1. Komputer: Merupakan perangkat keras yang digunakan untuk menjalankan perangkat lunak atau aplikasi dan menyimpan data yang dibutuhkan. Contoh dari komputer adalah desktop, laptop, dan tablet.

2. Sistem Operasi: Merupakan perangkat lunak yang dibutuhkan komputer untuk menjalankan perintah-perintah dan aplikasi. Contoh sistem operasi adalah Windows, macOS, dan Linux.

3. Aplikasi perkantoran: Perangkat lunak yang digunakan untuk menyelesaikan tugas-tugas perkantoran. Contohnya adalah Microsoft Office (Word, Excel, PowerPoint), Google Docs, dan Libre Office.

4. Internet dan Intranet: Merupakan koneksi jaringan yang memungkinkan pengguna untuk mengakses informasi dari berbagai tempat atau membagikan informasi dengan pengguna lain. Contoh dari internet adalah web browser seperti Google Chrome, Mozilla Firefox; sedangkan contoh dari Intranet adalah SharePoint dan Jira.

5. Perangkat telekomunikasi: Perangkat keras dan lunak yang digunakan untuk berkomunikasi jarak jauh, seperti telepon genggam, telepon rumah, dan aplikasi chatting seperti Whatsapp, Telegram, dan Skype.

6. Aplikasi pengolah gambar dan video: Perangkat lunak yang digunakan untuk mengedit, memodifikasi, dan memanipulasi gambar dan video. Contoh software pengolah gambar dan video iyalah Adobe Photoshop, CorelDRAW, dan Adobe Premiere Pro.

7. Aplikasi pengolah audio: Perangkat lunak yang digunakan untuk merekam, memodifikasi, dan mengedit file suara. Contoh software pengolah audio seperti Adobe Audition, AudaCity.

16. sebutkan kegunaan Directx diagnostic tool


DirectX memiliki beberapa komponen yang saling melengkapi untuk dapat mengoptimalkan kinerja hardware, diantaranya :


1. DirectX Graphics.
merupakan komponen dalam directx yang menangani masalah pada graphic, biasanya sangat dibutuhkan oleh games yang akan kita mainkan. mulai dari direct draw ( graphic 2 dimensi ) sampai direct3d ( graphic 3 dimensi.

2. DirectX Input.

komponen pada directx yang menangani bagian peripheral hardware input, seperti mouse, keyboard, joystick, dll.

3. DirectX Sound
komponen dasar yang berisi konfigurasi audio pada komputer. banyak media player yang mewajibkan adanya direct sound ini seperti misalnya VLC player dan WinAMP

2.Virtual memory Berfungsi untuk membantu kinerja RAM dengan membuat sebuah virtual RAM pada harddisk computer

3.system properties - basic information about computer berfungsi sebagai pemandu user untuk mengetahui dasar - dasar dari penggunaan computer.

4.disk management berfungsi sebagai pengkonfigurasi partisi harddisk

5.directx diagnostic digunakan untyuk mendiagnosa  dan memperbaiki directx.

6.windows task manager berfungsi sebagai pengatur program - program dan services yang berjalan di windows.
