2. pengertian dan fungsi unix


Pengertian UNIX adalah suatu sistem operasi yang memiliki model client-server yang mendukung untuk digunakan dalam jaringan computer karena keunggulannya yang dapat melakukan fungsi utamanya jaringan komputer lebih baik dari sistem operasi lain. Sistem operasi ini dikembangkan oleh satu bagian di AT&T.  

Fungsi Unix antara lain :

- Untuk multi-tasking

- Untuk multi-user / timesharing

- Untuk multi-programming

- Untuk multi-processing

Pembahasan

Sistem unix memiliki konsep penyimpanan berjenjang dengan media teks biasa untuk penyimpanan data.dan keunikannya adalah perangkat yang terhubung mendapat perlakuan seperti file. Sistem operasi ini cenderung stabil sehingga dijadikan percontohan sistem operasi masa sekarang. UNIX menggunakan CLI dalam mengeksekusi perintah – perintah di dalam sistemnya

Terdapat 3 komponen yang ada pada UNIX  diantaranya :

Kernel Development Environment Commands

Dalam menghubungkan antar sistem computer, UNIX memerlukan dua media utana yakni Ethernet dan serial line internet protocol yang dapat dikembangkan menjadi point to point protocol

UNIX membagi akses menuju direktori di dalam sistemnya  menjadi taiga yaitu

User/Owner Groups Other

Kita juga sering mendengar hubungan antara UNIX dengan Linux. Keduanya adalah sistem operasi namun Linux dapat dikatakan sebagai kloningan dari UNIX ataupun bagian dari sistem UNIX.

Fungsi unix penjabarannya tidak jauh berbeda dengan linux karena linux sendiri adalah clone dari unix diantaranya :

Multi-tasking

Sebagai sistem operasi yang dikendalikan oleh penggunanya, sistem operasi dapat menjalankan program - program secara bersamaan tanpa menimbulkan masalah.

Multi-programming

Seperti yang kita tahu, fungsi sistem operasi sendiri adalah mengatur manajemen resources perangkat berserta operasinya sehingga fungsi ini membuat sistem operasi dapat mengakses dan melakukan tindakan - tindakan atas resources ini dalam waktu yang sama. Sistem unix ini juga sangat digemari oleh para programmer.

Multi-Processing

sistem operasi unix dapat menjalankan processor pada beberapa cpu dalam waktu yang sama

Multi-user / Time sharing

Dengan adanya sistem operasi unix, dapat berjalan model multiuser yang menggunakan cpu secara serentak dan secara otomatis akan diatur secara otomatis pembagiannya.
