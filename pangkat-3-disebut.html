25. Bilangan Berpangkat Memahami konsep bilangan berpangkat Pertanyaan: bagaimana kalian dapat menggunakan bentuk pangkat untuk menyederhanakan penulisan sebuah bilangan? Contoh: 3 x 3 x 3 x 3 x 3 x 3 = 36 dapat disebut dengan perpangkatan 3 (-2) x (-2) x (-2) x (-2) = (-2)4 dapat disebut perpangkatan -2 Secara umum perkalian berulang dari suatu bilangan a dapat disebut dengan perpangkatan a


Kita ingat kembali mengenai eksponen. Eksponen merupakan suatu perkalian berulang. Biasanya eksponen merupakan bilangan yang sama dikali berulang sebanyak [tex] n [/tex] kali.

Misalkan [tex] a [/tex] adalah bilangan sembarang. Eksponen dapat ditulis sebagai

[tex] a^n = \underset{n \text{ kali}}{\underbrace{a \times \dots \times a}} \\ [/tex]

Sebagai contoh, [tex] 2^2 = 2 \times 2 [/tex] yang berarti bilangan [tex] 2 [/tex] dikalikan sebanyak [tex] 2 [/tex] kali. Contoh lainnya [tex] (-1)^4 = -1 \times (-1) \times (-1) \times (-1) = 1 [/tex] yang berarti sebanyak empat kali bilangan [tex] (-1) [/tex] dikalikan.

Seperti yang kita lihat di pertanyaan (untuk contoh) lainnya), [tex] 3 \times 3 \times 3 \times 3 \times 3 \times 3 [/tex] dapat dinotasikan sebagai [tex] 3^6 [/tex] dan tentu saja dengan [tex] -2 \times (-2) \times (-2) \times (-2) = (-2)^4 [/tex].
