14. umur pion jika berada di laboratorium adalah 4 mikrosekon, jika pion tersebut bergerak dengan kecepatan 0,6 C MAKA UMURNYA MENJADI ........(mikrosekon)


diket:

t₀ = 4 μs = 4 x 10⁻⁶ s

kecepatan (v) = 0,6 c

ditanya:

t....?

jawab:

[tex]t = \frac{t_0}{\sqrt{1 - \frac{v^2}{c^2} } }\\[/tex]

[tex]t = \frac{4 \times 10^{-6}}{\sqrt{1 - \frac{(0,6c)^2}{c^2} } }\\[/tex]

  [tex]= \frac{4 \times 10^{-6}}{\sqrt{1 - \frac{0,36c^2}{c^2} } }\\[/tex]

  [tex]= \frac{4 \times 10^{-6}}{\sqrt{1 - 0,36}}\\[/tex]

  [tex]= \frac{4 \times 10^{-6}}{\sqrt{0,64}}\\[/tex]

  [tex]= \frac{4 \times 10^{-6}}{0,8}\\[/tex]

  [tex]= 5 \times 10^{-6} \: s[/tex] = 5 mikrosekon

Jadi, umur pion menjadi 5 mikrosekon.

Penjelasan Dengan Langkah-langkah

Operasi hitung bilangan bulat adalah suatu bilangan yang meliputi penjumlahan, pengurangan, perkalian, dan pembagian. Persis sekali dengan namanya, bilangan bulat positif adalah suatu bilangan positif di sebelah kanan nol pada sebuah garis bilangan. Misalnya 1,2,3,4,5,6,7,8,9, dan seterusnya. Sebaliknya, bilangan bulat negatif adalah suatu bilangan negatif yang berada di sebelah kiri nol pada sebuah garis bilangan. Misalnya -1,-2,-3,-4,-5,-6,-7,-8,-9, dan seterusnya. Pada operasi hitung bilangan bulat, terdapat beberapa cara atau rumus untuk menghitung. Berikut rumusnya:

Rumus Penjumlahan bilangan bulat:

(+) + (+) = (+)(+) + (-) = (+) / (-)(-) + (-) = (-)(-) + (+) = (+) / (-)

Rumus pengurangan bilangan bulat:

(+) - (+) = (+) / (-)(+) - (-) = (+)(-) - (-) = (+) / (-)(-) - (+) = (-)

Rumus perkalian bilangan bulat:

(+) x (+) = (+)(-) x (-) = (+)(+) x (-) = (-)(-) x (+) = (-)

Rumus pembagian bilangan bulat:

(+) : (+) = (+)(-) : (-) = (+)(+) : (-) = (-)(-) : (+) = (-)

Agar lebih mengerti dan paham mengenai operasi hitung bilangan bulat, kakak akan memberikan contoh soal dari materi ini, yaitu operasi hitung bilangan bulat. berikut contoh soalnya:

Contoh operasi hitung penjumlahan bilangan bulat.

Positif dengan positif

4 + 8 = 1215 + 10 = 2514 + 16 = 30

Negatif dengan negatif

(–14) + (–6) = –(14 + 6) = –20(–15) + (–15) = –(15 + 15) = –30(–14) + (–16) = –(14 + 16) = –30

Negatif dengan positif

(–14) + 6 = 6 – 14 = –12(–15) + 15 = 15 – 15 = 0(–14) + 16 = 16 – 14 = 2

Positif dengan negatif

14 + (–6) = 14 – 6 = 815 + (–15) = 15 – 15 = 014 + (–16) = 14 – 16 = –(16 – 14) = –2
