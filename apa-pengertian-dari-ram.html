9. Apakah kepanjangan dari RAM..?? Dan Apakah itu RAM?


RAM (Random Access Memory)

Merupakan jenis memori yang isinya dapat diganti-ganti selama komputer dihidupkan sebagai suatu penyimpanan data yang dapat dibaca atau ditulis dan dapat dilakukan secara berulang-ulang dengan data yang berbeda-beda. Random artinya data yang disimpan pada RAM dapat diakses secara acak. Modul memori RAM yang umum diperdagangkan berkapasitas 128 MB, 256 MB, 512 MB, 1 GB, 2 GB, dan 4 GB

RAM dibagi lagi menjadi dua jenis, yaitu jenis statik dan dinamik.

RAM statik menyimpan satu bit informasi dalam sebuah flip-flop. RAM statik biasanya digunakan untuk aplikasi-aplikasi yang tidak memerlukan kapasitas memori RAM yang besar.RAM dinamik menyimpan satu bit informasi data sebagai muatan. RAM dinamik menggunakan kapasitas gerbang substrat sebuah transistor MOS sebagai sel memori elementer.

29. jelaskan Jelaskan perbedaan rom dan ram serta jelaskan pengertian masing-masing ​


Jawaban:

Definisi :

RAM

Singkatan dari Random Access Mermory adalah sebuah hardware untuk penyimpanan sementara program & data program pada saat menjalaskan aplikasi atau program. Di karenakan sementara atau Volatile jadi jika PC/Laptop mati data pada RAM akan kosong.

ROM

Singkatan dari

Read Only Memory adalah sebuah hardware chip memory semi konduktor yang terletak pada Mobo<Mother Board>. Biasanya ini digunakan untuk menyimpan data atau frimware <BIOS/UEFI> default bawaan pabrik yang bisa di baca oleh perangkat. ROM termasuk memory non-volatile jadi data tidak akan hilang jika PC/Laptop di matikan.

Jenis-Jenis

RAM :

1. D RAM <Dynamic RAM>

     RAM yang di refresh oleh CPU <Prosesor> agar data-data didalamnya tidak hilang, kekurangannya hanya bisa menyimpan sedikit data

2. SD RAM < Sychronous Dynamic RAM>

   teknologi lanjutan dari D RAM, SD RAM sudah di sinkronisasikan oleh clock system dan lebih cepat dari D RAM

3. S RAM <Static RAM>

   RAM ini berbeda dari teknologi terdahulunya, RAM jenis ini dapat menyimpan data tanpa harus di refresh dari CPU <Prosesor> dan bekerja lebih cepat dari D RAM & SD RAM

4. DDR RAM < Double Rate RAM>

   RAM ini yang sering kita gunakan di PC/Laptop kita, Teknologi ini lebih maju dari yang lainnya dan terus memunculkan generasi terbaru seperti DDR 2, DDR 3, DDR4 <yang kita gunakan>, DDR 5 <Rencana dan sudah ada beberapa mobo yg ada dengan teknologi ini>

5. RDR RAM <Rambus Dynamic Access Memory>

  RAM ini digunakan pada system Pentium 4 dan bekerja lebih cepat dari SD RAM dan Lebih mahal.

6. EDO RAM < Extended Data Out RAM>

  RAM untuk OS Pentium, dan mampu menyimpan sekaligus mengambil data bersamaan

ROM

1. Stock ROM

   Frameware bawaan dari pabrik dan bersifat default, standar dan belum dimodif

2. Custom ROM

   Frameware yang sudah dimodifikasi

3. Manufacturer < Carrier Stock ROM>

   Frameware yang di modif oleh pabrik agar bisa di costum pada aplikasi

4. Mask ROM

   Data ditulis saat chip di buat

5. PROM <Programmable Read Only Memory>

   Data ditulis setelah chip di buat

6. EPROM <Eraseable Programmable Read Only Memory>

   bisa di hapus sengan sinar UV yang sangat kuat

7. EEPROM <Electrically Erasable Programmable Read Only Memory>

  data bisa dihapus dengan teknolofi elektrik seperti emisi elektron

Fungsi

RAM

1. Membaca Data

     membaca data dari HDD/SDD/SSHD  pada saat dibuka

2. Penyimpan Sementara

   data-data program/aplikasi pada saat berjalan

ROM

Penyimpanan frimware <BIOS/UEFI> pada Mobo<mother board>

Cara Kerja

RAM

Bekerja pada saat program/ aplikasi dalam PC/Laptop di jalan kan

untuk Windows  untuk melihat apa saja aplikasi/program yg sedang di jalankan oleh user dan system dapat di lihat di task manager

ROM

Hanya dapat di baca oleh user. Karena data-data sofware secara default adalah manufaktur dari Mobo <Mother Board> tersebut.

1.Pengertian Modem

Modem merupakan singkatan dari Modulator Demodulator. Dimana kedua kata ini memililiki arti yaitu Modulator merupakan bagian yang berfungsi untuk mengubah sinyal informasi menjadi sinyal pembawa yang siap dikirimkan, sedangkan arti dari Demodulator merupakan bagian untuk memisahkan antara sinyal informasi dari sinyal pembawa yang diterima dengan baik. Dengan kata lain, modem adalah jenis alat komunikasi dua arah.

Macam Jenis modem

Macam Jenis Tipe Modem Terbaik

⊃ Modem Eksternal

Modem eksternal adalah sebuah alat yang digunakan untuk komputer agar dapat mengakses internet dimana data yang berasal dari komputer yang berbentuk sinyal digital akan diubah menjadi sinyal analog. Sehingga jika modem menerima data yang berupa sinyal analog akan diubah menjadi sinyal digital kemudian akan diproses lebih kanjut oleh komputer. Sinyal analog ini dapat dikirimkan menggunakan media telekomunikasi seperti telepon dan radio.

⊃ Modem Internal

Jika data tersebut sudah tiba pada modem yang dituju kemudian sinyal analog ini akan diubah kembali menjadi sinyal digital dan langsung dikirimkan ke komputer.

⊃ Modem ISDN

Modem ISDN berasal dari singkatan Integrated Services Digital Network adalah sebuah sistem telekomunikasi yang terdapat dalam layanan berupa data, suara, atau gambar yang kemudian akan diintegrasikan ke dalam suatu jaringan, dimana jaringan ini menyediakan konektivitas digital dari ujung ke ujung untuk menunjang ruang lingkup pelayanan yang luas.

⊃ Modem GSM

Modem GSM adalah jenis modem yang menggunakan telepon seluler sebagai media sistem untuk transfer data. Modem GSM biasanya didukung dengan menggunakan kartu koneksi yang mendukung provider yang memiliki sifat GSM.

⊃ Modem Analog

Seperti namanya fungsi dari modem analog ini adalah sebagai alat untuk membentuk sinyal analog kemudian diubah menjadi sinyal digital

⊃ Modem ADSL

Modem ADSL berasal dari singkatan Asymetric Digital Subscribe Line merupakan alat untuk mengakses internet dan menggunakan telepon analog secara bersamaan. Dimana cara penggunaannya dibantu menggunakan alat penghubung yang disebut Splitter. Fungsi splitter pada ADSL adalah untuk menghilangkan gangguan sehingga hasilnya dapat berjalan secara bersamaan, yakni dapat menggunakan internet dan juga dapat menggunakan telepon biasa. Tapi tidak saling bertabrakan.

⊃ Modem Kabel

Modem kabel adalah jenis modem yang digunakan untuk menerima data secara langsung dari suatu penyedia layanan melalui TV Kabel.

⊃ Wireless modem

Wireless atau dalam bahasa indonesia disebut nirkabel, adalah teknologi yang menghubungkan dua piranti untuk bertukar data tanpa media kabel. Cara Kerja wireless modem : Data dipertukarkan melalui media gelombang cahaya tertentu (seperti teknologi infra merah pada remote TV) atau gelombang radio (seperti bluetooth pada komputer dan ponsel) dengan frekuensi tertentu.

⊃ Modem CDMA

Modem CDMA adalah jenis modem yang menggunakan frekuensi CDMA 800 MHz atau CDMA 1x. Bahkan modem CDMA yang terbaru saat ini sudah menggunakan frekuensi EVDO Rev-A atau setara 3G dan yang paling terbaru bahkan sedang berkembang adalah modem CDMA dengan frekuensi EVDO Rev-B.

Fungsi Modem

Secara umum, fungsi modem adalah sebagai hardware atau perangkat keras yang berguna untuk mengubah komunikasi 2 arah yaitu mengubah sinyal digital menjadi sinyal analog dan sebaliknya. Fungsi modem lainnya yaitu :

Alat pengubah sinyal digital menjadi sinyal analog

Menghubungkan perangkat dalam jaringan

Melakukan kegiatan modulasi dan demodulasi

Melakukan pemeriksaan paket data dan komunikasi

Melakukan kompres data yang dikirimkan melalui sinyal

1.PROCESSOR

Komputer adalah alat modern yang terbentuk dari komponen - komponen perangkat keras (hardware) dan perangkat lunak (software). Salah satu perangkat keras yang ada pada komputer adalah processor

2.-Control unik yang dimana adalah bagian yang bekerja untuk melakukan pengontrolan terhadap tugas yang dimiliki oleh sebuah komputer. Dan bagian dari control unit ini memiliki sebuah peranan yang dimana sangatlah penting didalam melakukan pengontrolan daripada seluruh perangkat yang dimana akan terhubung dengan sebuah komputer. Fungsi yang dimiliki oleh control unit adalah untuk melakukan penyesuaian dari seluruh perangkat yang ada.

-Arithmetic logic unit yang dimana adalah bagian dari processor yang memiliki fungsi untuk menjadi bagian yang melakukan seluruh perhitungan daripada operasi komputer. Arithmetic logic unit ini sendiri akan dapat melakukan pengerjaan tugas perhitungan dengan menggunakan aritmatika dan juga logika.

-Register yang dimana adalah alat untuk melakukan penyimpanan yang dimana memiliki sebuah kecepatan yang sangatlah tinggi. Selain itu, register memiliki fungsi untuk melakukan penyimpanan dari seluruh perintah yang pada saat itu sedang diproses oleh komputer.

-CPU Interconnection yang dimana memiliki fungsi untuk menghubungkan bagian Control Unit, Arithmetic Logic Unit, Register dan juga berbagai macam sistem lainnya.

3.RAM adalah berasal dari singkatan Random Access Memory, RAM yaitu suatu memori tempat penyimpanan data sementara, ketika komputer dijalankan dan dapat diakses secara acak (random).

Fungsi RAM adalah mempercepat pemeproses data pada PC atau komputer. 

4.1.DRAM (Dynamic RAM) adalah jenis RAM harus sering di refresh oleh CPU agar data yang terkandung didalamnya tidak hilang.

2.SDRAM (Synchronous Dynamic RAM) adalah jenis RAM yang paling umum digunakan pada PC Pentium I,II dan III. RAM ini disinkronisasi oleh clock sistem dan memiliki kecepatan lebih tanggi dari pada DRAM serta dapat digunakan teritama dalam cache.

3.SRAM (Statik RAM) adalah jenis memory yang tidak perlu penyegaran oleh CPU agar data yang terdapat didalamnya tetap tersimpan dengan baik.

4.RDRAM (Rambus Dynamic RAM) adalah pada jenis memory ini yang lebih cepat dan lebih mahal dari pada SDRAM. Memory ini bisa digunakan pada sistem yang menggunakan Pentium 4.

5.FPM DRAM (Fast Page Mode DRAM) adalah merupakan bentuk asli dari DRAM. Laju transfer maksimum untuk cache L2 mendekati 176 MB per sekon.

 6.EDO DRAM (Extented Data Out DRAM) adalah memory ini sekitar 5% lebih cepat dibandingkan dengan FPM. Laju transfer maksimum untuk cache L2 mendekati 264 MB per sekon.

7.Double Data Rate RAM (DDR RAM), dinamis untuk generasi Pentium IV (DDR 400). Memiliki banyak tipe dan umumnya dikelompokkan berdasarkan bandwidth dan dipengaruhi besarkan clock yang diberikan memori itu nantinya. Tipe yang ditawarkan PC 1600 / 2100 / 2700 / Untuk pemilihan jenis memori sangat dipengaruhi kinerja processor, karena perbedaan antar internal bus pada processor dengan internal bus pada memori akan berakibat fatal bagi kecepatan proses pengolahan aplikasi data, untuk itu perbandingan keduanya haruslah sebanding
