3. Sebutkan sistem operasi yg tepat untuk jaringan​


Jawaban:

Suatu jaringan komputer bisa terdapat:

1 client & 1 server maupun dapat berupa lebih dari 1 client & 1 server.

Contoh sistem operasi untuk komputer client antara lain:

Linux Knoppix, Linux Mandrake, Linux Zorin, Windows XP, Windows 7, Windows 8.

Contoh sistem operasi untuk komputer server antara lain:

Linux Debian, Linux Ubuntu Server, Linux Fedora Server, Windows Server 2003, Windows Server 2008, Windows Server 2012

Penjelasan:

NOS (Network Operating System) termasuk salah 1 hal yg wajib ada dalam suatu jaringan. Yang perlu diperhatikan yaitu pemilihan sistem operasi yg tepat sesuai kebutuhan (untuk client dan untuk server).
