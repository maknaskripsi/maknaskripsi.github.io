5. harga 3 pasang sepatu dan 4pasang sendal Rp 280.000,00, sedangkan harga 4pasang sepatu dan 3pasang sendal Rp 315.000,00. harga sepasang sepatu dan 2pasang sendal adalah...


3x + 4y = 280.000
4x + 3y = 315.000

eliminasi
3x + 4y = 280.000 (x 3)
4x + 3y = 315.000 (x 4)
9x + 12y = 840.000
16x + 12y = 1.260.000 _
- 7x = - 420.000
x = - 420.000 / -7
x = 60.000

 substitusi
3x + 4y = 280.000
3 (60.000) + 4y = 280.000
180.000 + 4y = 280.000
4y = 280.000 - 180.000
4y = 100.000
y = 100.000 / 4
y = 25.000

sepasang sepatu dan 2 pasang sendal
= x + 2y
= 60.000 + 2 (25.000)
= 60.000 + 50.000
= 110.000

jadi harga sepasang sepatu dan 2 pasang sendal adalah rp. 110.000
spatu = u
sandal = l
3u + 4l = 280.000 dikalikan 4
4u + 3l = 315.000 dikalikan 3
menjadi
12u + 16l = 1.120.000
12u + 9l  = 945.000    _ eliminasi
         7l  = 1750.000
           l = 25 .000
subtitusikan
3u + 4l = 280.000
3u + 4(25.000) = 280.000
3u + 100.000 = 280.000
       3u      = 280.000-100.000
       3u      = 180.000
         u      = 60.000

jadi hasil dari 1 spatu dan 2 sandal
 u + 2(l) = 60.000 + 2(25.000)
             = 60.000 + 50.000
             = 110 .000
jawabnnya adalah 110.000

38. 3. Suatu toko sepatu menjual 10 pasang sepatu laki laki dan 15 pasang sepatu perempuan seharga Rp 10.500.000,00. Jika terjual lagi 20 pasang sepatu laki laki dan 15 pasang sepatu perempuan seharga Rp 13.500.000,00. Harga sepasang sepatu laki - laki dan sepasang sepatu perempuan adalah.... Butuh secepatnyaa !!


Jawaban:

sepatu laki-laki = 300.000

sepatu perempuan = 500.000

Penjelasan dengan langkah-langkah:

misal :

sepatu laki = x

sepatu perempuan = y

diketahui :

10x+15y = 10.500.000...pers 1

20x+15y = 13.500.000...pers 2

eliminasi nilai y di pers 1 dan pers 2

10x+15y = 10.500.000

20x+15y = 13.500.000 -

-10x = -3.000.000

x = -3.000.000/-10

x = 300.000

substitusi x = 300.000 ke pers 1

10x+15y = 10.500.000

10(300.000)+15y = 10.500.000

3.000.000+15y = 10.500.000

15y = 10.500.000-3.000.000

15y = 7.500.000

y = 7.500.000/15

y = 500.000

jadi harga sepasang sepatu laki - laki dan sepasang sepatu perempuan berturut-turut adalah 300.000 dan 500.000
