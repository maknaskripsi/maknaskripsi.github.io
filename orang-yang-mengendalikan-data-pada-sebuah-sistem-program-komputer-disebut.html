11. Program yang mengendalikan operasi dari hardware komputer untuk melakukan proses pengubahan data menjadi informasi dikenal dengan sebutan


Program yang mengendalikan operasi dari hardware komputer untuk melakukan proses pengubahan data menjadi informasi dikenal dengan sebutan sistem operasi. Sistem operasi memberikan perintah dari hardware untuk menghasilkan data.

Pembahasan

Sistem operasi komputer adalah program perangkat lunak atau perangkat lunak yang diperlukan untuk mengelola dan mengoperasikan perangkat komputasi seperti komputer, smartphone, atau tablet. Sistem operasi berperan sebagai penghubung perangkat lunak (software) dan perangkat keras (hardware), dan dapat dikatakan sebagai pengelola sumber daya pada peralatan komputer. Dengan kata lain, sistem operasi bekerja untuk mengontrol fungsi-fungsi perangkat keras seperti memori, CPU, dan hard disk, dan untuk mengontrol fungsi-fungsi program perangkat lunak yang terhubung ke perangkat keras. Semua fungsi dan tujuan dari sebuah sistem operasi komputer adalah tujuan utama pengguna ketika menggunakan komputer.
