5. untuk melakukan perawatan PC menggunakan disk cleanup yang ada pada windows, file-file yang tidak diperlukan dapat dihapus dengan cara


Pada Windows 7, kamu bisa klik Start > All Programs > accessories > System tools > Disk Cleanup
pilih partisi yg akan di bersihkan dari sampah (bukan format/hapus semua data ya).
cthnya drive (c), nanti akan ada proses scanning file sampah. nanti akan muncul lokasi file sampah, pilih dengan tanda centang untuk menghapusnya, lalu klik OK dan pilih Delete Files. Selesai deh :D

9. 1.Apa Yang Disebut Dengan Perawatan Hardware..??2.Apa Yang Disebut Dengan Perawatan Software..??​


Jawaban dan Penjelasan:

1. Hardware adalah komponen perangkat keras, yang terdapat pada suatu perangkat komputer. Fungsi hardware sendiri adalah, sebagai media pengolahan data yang diinput oleh operator. Untuk selanjutnya data akan diproses menjadi data output yang berupa informasi.

2. Software adalah perangkat lunak, istilah ini digunakan untuk data yang telah diformat, dan disimpan ke dalam media penyimpanan dalam bentuk digital. Berbagai macam program yang terinstal di dalam perangkat komputer, juga masuk dalam kategori software. Termasuk juga sistem operasi, yang digunakan pada komputer tersebut.
