Jawaban:

a.=16m/sb.=32m

Penjelasan:

Rumus GLBB.

St=Vo•t+1/2•a•t²

Vt=Vo+a•t

Diketahui

Vo=0m/s

a=4m/s²

t=4s

ditanya St dan Vt.

a. Vt=Vo+a•t

Vt=0+4•4

Vt= 16 m/s

b. St =Vo•t+1/2•a•t²

St=0•4+1/2•4•4²

St=32 m

Seperti namanya, prinsip dari kereta api ini adalah memanfaatkan gaya magnet untuk mengangkat kereta sehingga mengambang, tidak menyentuh rel sehingga gaya gesek dapat dikurangi.
