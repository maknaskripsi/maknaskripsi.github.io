V. kerucut + V. tabung

= 4.106,6 cm³

1. lingkaran

2. persegi

K = 2 x (p + l)

L = p x l

4. segitiga

K = sisi 1 + sisi 2 + sisi 3

L = ½ x a x t

5. layang layang

K = sisi a + b + c + d

L = ½ x diagonal 1 x diagonal 2

6. belah ketupat

K = 4 x s

L = ½ × d1 × d2

7. jajar genjag

K = 2 x panjang + 2 x lebar atau bisa ditulis dengan 2 (panjang + lebar)

9. kubus

L = (2 x luas alas) + (keliling alas x tinggi prisma)

V = luas alas x t prisma

13. tabung

L = 2 x π x r (r + t)

V = π x r² x t

14. kerucut

L = π x r x (r + s)

V = 1/3 x luas alas x t
