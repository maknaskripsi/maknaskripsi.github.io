Penyelesaian

1.) Sebenarnya, 2 pertandingan ini adalah Babak Final dan Perebutan Juara 3, jadi tidak menggunakan sistem poin.

Argentina ( 1 ) vs ( 0 ) Brazil = ( Argentina menang, mendapat 3 poin )Kolombia ( 3 ) vs ( 2 ) Peru = Kolombia menang, mendapat 3 poin )

Kesimpulannya :

Timnas ArgentinaTimnas Kolombia

__________________

2.) Tinggal menjumlahkan jumlah golnya dalam 2 pertandingan :

= 3 ( Kolombia ) + 2 ( Peru ) + 1 ( Argentina )

= 5 + 1

= 6

__________________

3.) Tim bermain sejak menit 0, maka tinggal mengurangi menit pertandingan :

= 23 - 1

= 22 menit
