27. Tentukan kadar nitrogen dalam pupuk ZA (mengandung (NH4)SO4) yang memiliki kemurnian 95% jika dikehatui ArN=14, ArH=1, ArS=32, ArO=16


Jawaban :

Kadar Nitrogen dalam pupuk ZA Mengandung [(NH4)2 SO4] yang memiliki kemurnian 95% adalah Sebanyak 20,15 %

Penjelasan :

Diketahui

Kemurnian Pupuk = 95 %

Ar N = 14

Ar H = 1

Ar S = 32

Ar O = 16

Ditanya : Kadar nitrogen dalam pupuk ZA .............?

Jawab :

Langkah Pertama adalah Mencari Jumlah Mr [(NH4)2 SO4]

Mr [(NH4)2 SO4] = 2.N + 8.H + 1.S + 4.O

Mr [(NH4)2 SO4] = 2.14 + 8.1 + 1.32 + 4.16

Mr [(NH4)2 SO4] = 28 + 8 + 32 + 64

Mr [(NH4)2 SO4] = 132

Langkah Terakhir adalah Mencari Kadar Nitrogen dengan menggunakan Rumus :

% = ( Jumlah Ar N/Mr [(NH4)SO4] ) × Kemurnian Pupuk

% = ( 28/132 ) × 95 %

% = 0,21 × 95 %

% = 20,15 %
